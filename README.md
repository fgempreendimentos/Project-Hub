# Smart Affiliate Bot

Sistema de automação para divulgação de ofertas utilizando links de afiliados.

## Objetivo

Automatizar a busca, validação e publicação de promoções em grupos do WhatsApp utilizando meus links de afiliado.

O sistema será capaz de:

- Buscar promoções automaticamente
- Validar descontos
- Evitar promoções duplicadas
- Gerar textos utilizando IA
- Converter links para afiliado
- Publicar automaticamente no WhatsApp
- Registrar estatísticas
- Disponibilizar um Dashboard Web
